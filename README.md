## icecube_data_reader

[![codecov](https://codecov.io/github/nu-isance/icecube_data_reader/branch/master/graph/badge.svg?token=G1SR407LQU)](https://codecov.io/github/nu-isance/icecube_data_reader)

A collection of classes to load IceCube track data. Not affiliated with The IceCube Collaboration.

This code is based upon previous implementations found in [icecube_tools](github.com/nu-isance/icecube_tools), written by [Francesca Capel](https://github.com/cescalara) and [Julian Kuhlmann](https://github.com/specktakel), 
as well as [hierarchical_nu](https://github.com/nu-isance/hierarchical_nu/) ([Capel et al. 2025](https://doi.org/10.3847/1538-4357/ad7fe9)).
